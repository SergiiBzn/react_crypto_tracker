# Crypto Tracker

Crypto Tracker is a React-based single-page application for monitoring cryptocurrency prices and market data in real time. The app uses the CoinGecko API to fetch live market information and presents it with a clean, modern light UI.

## Features

- Browse a list of top cryptocurrencies with key metrics
- Search by coin name or symbol
- Sort by rank, name, price, or market cap
- Switch between grid and list views
- View detailed information for each coin (price, 24h change, market cap, volume, supply, and 7-day price chart)
- Responsive layout optimized for desktop and mobile

## Tech Stack

- React
- Vite
- React Router
- Recharts
- CoinGecko API

## Getting Started

```bash
npm install
npm run dev
```
